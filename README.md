# 💳 SpendDNA - Your Wallet's Year-End Story

SpendDNA is a Python-based financial analytics project that analyzes six months of UPI/bank transaction data and generates a detailed spending report. It cleans messy transaction records, extracts vendors, categorizes expenses, detects anomalies, identifies spending archetypes, and provides meaningful financial insights.

## 📌 Project Overview

This project is built using Python, Pandas, and NumPy without using Machine Learning or visualization libraries. It processes raw transaction data and produces a clean console report similar to a Spotify Wrapped for your wallet.

## 🚀 Features

- Transaction Parsing and Data Cleaning
- Vendor Extraction from Transaction Descriptions
- Spending Category Classification
- Executive Spending Summary
- Top Categories and Vendors
- Monthly Spending Trend Analysis
- Time-of-Day Spending Analysis
- Z-Score Based Anomaly Detection
- Spending Archetype Detection
- Financial Insights Report
